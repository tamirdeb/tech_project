# Wikipedia Automation Task

A lightweight test automation framework built to validate UI and API data integrity on Wikipedia. 

## Tech Stack
* **Language:** C# (.NET 8)
* **UI Automation:** Playwright
* **API Testing:** Native `HttpClient` (MediaWiki API)
* **Test Runner:** NUnit
* **Reporting:** ExtentReports
